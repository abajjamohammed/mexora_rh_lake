# Mexora RH Intelligence — Data Lake & Analyse du Marché IT

Ce projet implémente une architecture Data Lake (Bronze, Silver, Gold) pour analyser le marché de l'emploi IT au Maroc à partir de 5000 offres d'emploi.

## 🚀 Structure du Projet
- `data_lake/` : Répertoire racine contenant les données (Bronze, Silver, Gold).
- `pipeline/` : Scripts Python de traitement des données.
- `analysis/` : Notebooks et requêtes SQL DuckDB.
- `generate_data.py` : Script de génération des données brutes.

## 🛠️ Installation
1. Cloner le dépôt:
   git clone https://github.com/abajjamohammed/mexora_rh_lake

2. Installer les dépendances :
   
   pip install -r requirements.txt
